# 📊✨ Employee Attrition Dashboard (Power BI Project)

## 📁 Project Overview
A fully interactive and dynamic Employee Attrition Dashboard built using Microsoft
Power BI with slicers, charts, and DAX measures to explore and
visualize HR data in a clean and user-friendly way.

## 🛠 Tools & Technologies Used
- 📊 Microsoft Power BI Desktop
- 🔄 Data Modeling & Relationships
- 🎛️ Slicers & Filters (for interactivity)
- 📐 DAX Measures (Attrition Rate, Avg Salary, etc.)
- 🎨 Custom Formatting & UI Design
